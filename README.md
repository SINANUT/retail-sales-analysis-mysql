# 🛒 Retail Sales SQL Practice Project

## 📌 Overview
This project demonstrates SQL data analysis using a retail sales dataset.  
The goal of the project is to practice real-world SQL queries and extract meaningful business insights using MySQL.

---

## 🎯 Project Objectives
- Practice SQL querying skills
- Analyze sales performance over time
- Identify best-selling months
- Find top customers based on sales
- Perform category-wise customer analysis

---

## 🛠️ Tools Used
- MySQL
- MySQL Workbench
- SQL

---

## 📂 Dataset
The dataset contains retail transaction records including:

- Sale Date
- Customer ID
- Product Category
- Quantity Sold
- Total Sale Amount

Dataset is included in this repository for practice purposes.

---

## 📊 SQL Concepts Covered
- SELECT statements
- GROUP BY & Aggregations
- ORDER BY
- Subqueries
- Window Functions (RANK)
- Date Functions (EXTRACT)
- Data Analysis Queries

---

## 🔎 Analysis Performed
- Average sales per month
- Best-selling month in each year
- Top 5 customers by total sales
- Unique customers per category
- Sales trend exploration
